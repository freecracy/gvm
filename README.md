# 简介
go语言版本管理,自用,非侵入.

# 安装

```bash
curl https://raw.githubusercontent.com/freecracy/gvm/master/scripts/install | bash
```

# 使用
```bash
Use Age:
    gvm list                 列出已安装版本
    gvm listall              列出用版本
    gvm add [version]        安装对应版本
    gvm use [version]        选择对应版本
    gvm version              显示当前使用版本
    gvm rm [version|gvm] 卸载对应版本或卸载gvm
    gvm help                 显示帮助信息
```

# 卸载
```bash
gvm remove gvm
```
