# 简介

# 安装
## mac
```
# 暂未支持mac
brew tab install freecracy/gvm
```
## ubuntu
```bash
wget -q https://raw.githubusercontent.com/freecracy/gvm/master/gvm && sudo mv gvm /usr/local/bin/gvm && sudo  chmod +x /usr/local/bin/gvm 
```
# 使用
```bash
Use Age:
    gvm list
    gvm listall
    gvm add
    gvm use
    gvm version
    gvm remove
    gvm help
```

# 卸载
```bash
gvm remove gvm
```
